## 🚀 Featured Projects

### 🗺️ [London Synergy Index](https://github.com/daththeanalyst)
Predictive site selection across 55,000 H3 hexagons — integrating LandScan satellite rasters, ONS Census data and OSM POIs with graph-centrality features. Spatially cross-validated XGBoost classifier achieving **R² = 0.92** via Optuna, deployed with a 3D Pydeck/Streamlit dashboard.
`Python` `XGBoost` `H3` `NetworkX` `Streamlit` `AWS`

### ⚡ [RiskTerrain](https://github.com/daththeanalyst)
6-node **LangGraph** agentic pipeline ingesting live geopolitical events (USGS, NewsAPI), performing multi-hop supply chain traversal across 154 S&P 500 companies in **SurrealDB**, scoring cascading risk exposure in **under 8 seconds**. Hybrid graph-vector retrieval with sentence-transformer embeddings. Built at a LangChain/LangGraph hackathon.
`LangGraph` `LangChain` `SurrealDB` `RAG` `Docker` `Three.js`

### 🌐 [Dataportfolio.co.uk](https://dataportfolio.co.uk)
Production SaaS platform I designed, built and launched — full frontend and backend, LLM-backed APIs for CV extraction, a portfolio builder chat interface, and persistent data management. Zero code required for the end user.
`JavaScript` `Flask` `REST APIs` `LLMs` `Full-Stack`

### 🧹 [DataScrub](https://github.com/daththeanalyst)
Local AI data cleaning agent using **Ollama** — datasets never leave the device. Hybrid system combining LLM context understanding with rule-based cleaning logic (imputation, deduplication). Built for privacy-first environments.
`Ollama` `Local LLMs` `Python` `JavaScript`

### 🏠 [Geospatial London Housing & Crime Analysis](https://github.com/daththeanalyst)
Spatio-temporal pipeline merging 1M+ records to engineer a predictive Opportunity Index for investment strategy. XGBoost regressor tuned via Optuna, outperforming KNN and polynomial baselines.
`Python` `XGBoost` `Scikit-learn` `Optuna` `SHAP`
